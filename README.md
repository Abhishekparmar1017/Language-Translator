# Language Translator NLP Project

## Overview

This project is a simple Language Translator built using Python.  
It translates text from one language to another.

The project demonstrates the use of Natural Language Processing (NLP) for text translation.

## Features

- Translate text into multiple languages
- Simple command line interface
- Uses Python NLP libraries
- Beginner friendly project

## Technologies Used

Python  
Pandas  
Googletrans API  

## Project Structure

language-translator-nlp

data/ → dataset  
src/ → translation functions  
app.py → main program  
requirements.txt → project dependencies  

## Installation

Install 

pip install -r requirements.txt

Run the project

python app.py

## Example

Input

Hello how are you
